# About This Project

Welcome to the **Facet Filter Page**, a modern, high-performance product showcase designed for speed and flexibility. This application demonstrates an advanced faceted search system that allows you to explore datasets with ease.

## Key Features

Our filtering system is designed to be intuitive and powerful, offering several ways to find exactly what you're looking for:

-   **Full-Text Search:** Quickly locate items using the real-time search bar.
-   **Categorical Filtering:** Narrow down results by selecting one or more values from categories (e.g., color, material) using multi-select checkboxes.
-   **Range Sliders:** Use dual-handle sliders to filter by numeric ranges, such as price or dimensions.
-   **Boolean Switches:** Easily toggle binary states (e.g., "In Stock", "Compatible") with sleek on/off switches.
-   **Single-Handle Sliders:** Set minimum or maximum thresholds with continuous sliders, or use stepped sliders to snap to specific valid values.
-   **Reactive Interface:** Watch the product list and filter counts update instantly as you interact with the UI.
-   **Detailed Product View:** Click on any item to open a detailed modal with more information and high-quality images.

## Built for Flexibility

This project is **data-agnostic**, meaning it is not hard-coded to any specific set of products. It can be easily adapted to showcase any kind of inventory—from high-tech electronics to artisanal crafts.

### Easy Maintenance

The application is designed to be maintained without touching the source code:
-   **Edit via Spreadsheets:** Update your product list by simply editing a CSV file.
-   **Configure via YAML:** Define your filter groups and UI structure using human-readable YAML or CSV files.
-   **Instant Styling:** Change the theme colors, page title, and UI labels through a single configuration file.

## Open Source

This project is open-source and built using modern web technologies like **Vite**, **TypeScript**, **Bulma**, and **itemsjs**.

The source code is available on GitHub: [https://github.com/Kugeleis/facet-filter-page](https://github.com/Kugeleis/facet-filter-page)
